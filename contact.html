Frequently Asked Questions

Quick answers to common questions

How do I join the server?

To join our server, you need Minecraft Java Edition. Simply add our server IP address to your server list and connect. You can find the IP addresses on our Game Modes page.

What Minecraft version do you support?

We currently support Minecraft Java Edition versions 1.19 and above. We recommend using the latest stable version for the best experience.

How can I report a player?

If you encounter a player breaking the rules, you can report them using the /report command in-game, or contact our staff through Discord or email.

Do you offer refunds for shop purchases?

We offer refunds within 7 days of purchase if you're not satisfied with your purchase. Contact our support team with your order details to request a refund.
